# Food nutrition data

## Convert file format

Install [csvkit](https://csvkit.readthedocs.io/) to easy convert xls -> csv -> json

```bash
in2csv 食品營養成份資料庫\(2015版\).xls > 食品營養成份資料庫\(2015版\).csv
csvjson 食品營養成份資料庫\(2015版\).csv > 食品營養成份資料庫\(2015版\).json
```
